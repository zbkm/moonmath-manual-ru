= Руководство MoonMath

Руководство MoonMath — это ресурс для всех, кто хочет понять и раскрыть потенциал zk-SNARKs, от начинающих до экспертов.

Аббревиатура zk-SNARK расшифровывается как «Zero-Knowledge Succinct Non-Interactive Argument of Knowledge» (краткий неинтерактивный аргумент знания с нулевым разглашением) и относится к криптографической технике, позволяющей доказать корректное выполнение вычислений и владение определённой информацией, не раскрывая саму информацию. Zk-SNARKs, являясь разновидностью доказательств с нулевым разглашением (zero-knowledge proofs), открывают новую парадигму конфиденциальности и уже используются для обеспечения приватных транзакций в блокчейнах. Они также обладают потенциалом для безопасного масштабирования решений на основе блокчейна.

Большинство объяснений не способны прояснить, как и почему работают zk-SNARKs, поэтому они остаются чем-то «магическим» и неуловимым, отпугивая разработчиков от изучения и применения их в проектах. Ресурсы разбросаны по блогам и библиотекам GitHub. Это создаёт высокий порог входа, замедляя широкое распространение zk-SNARKs и связанных с ними технологий повышения конфиденциальности. Ещё хуже то, что внедрение zk-SNARKs без понимания фундаментальных основ их работы повышает риск проблем с безопасностью.

Мы хотим изменить это с помощью нашего Руководства MoonMath по zk-SNARKs, рассчитанного на аудиторию с минимальным опытом в криптографии и программировании.

Особенность Руководства MoonMath заключается в том, что оно позволяет читателям конструировать zk-SNARKs с помощью простых вычислений ручкой и бумагой. На протяжении всего руководства читателям предлагается выполнять упражнения, гарантирующие погружение в фундаментальные строительные блоки zk-SNARKs. При этом читатели создают минимально возможный, но полностью функциональный криптографический стек для вычисления внутренних компонентов zk-SNARK без помощи компьютера.

Проходя через руководство, читатели постигнут математические концепции, которые используются не только в SNARKs, но и в других доказательствах с нулевым разглашением и в криптографии вообще.

== Инструкции по сборке

=== Требования
* Docker

=== Локальная сборка

. Склонируйте репозиторий:
+
[source,bash]
git clone https://github.com/zbkm/moonmath-manual-ru.git
cd moonmath-manual-ru

. Запустите сборку последовательно:
+
[source,bash]
----
# Pass 1–2: генерация .sage и разрешение перекрёстных ссылок
docker run --rm -v "$PWD:/workdir" -w /workdir texlive/texlive:latest pdflatex --shell-escape main-moonmath.tex
docker run --rm -v "$PWD:/workdir" -w /workdir texlive/texlive:latest pdflatex --shell-escape main-moonmath.tex

# SageMath: вычисление блоков кода и графиков
docker run --rm \
  -v "$PWD:/workdir" \
  -v "$PWD/patches/sagetex.py:/home/sage/sage/local/var/lib/sage/venv-python3.12/lib/python3.12/site-packages/sagetex.py" \
  -w /workdir \
  sagemath/sagemath:latest \
  sage main-moonmath.sagetex.sage

# Pass 3–4: вставка результатов и финальные ссылки
docker run --rm -v "$PWD:/workdir" -w /workdir texlive/texlive:latest pdflatex --shell-escape main-moonmath.tex
docker run --rm -v "$PWD:/workdir" -w /workdir texlive/texlive:latest pdflatex --shell-escape main-moonmath.tex
----
